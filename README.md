# 音乐人格测试 · City Pop（lcl-mbti）

群周年庆限定的 MBTI 风格音乐人格测试 H5。答 10 道情境题 → 匹配 11 种音乐人格 → 生成分享图，截图丢群里对暗号。

- 纯前端、单文件、零外部依赖（微信内置浏览器友好，离线也能跑）。
- City Pop 霓虹复古风：合成波落日、透视网格地平线、扫描线、霓虹辉光。
- 分享图用原生 Canvas 画，长按即可保存。

## 本地预览

直接双击 `index.html` 用浏览器打开即可。或起个本地服务：

```bash
cd lcl-mbti
python3 -m http.server 5173
# 浏览器打开 http://localhost:5173
```

## 改文案 / 改题目

所有内容都在 `index.html` 里 `<script>` 顶部的「数据区」，改这两块就行，不用碰别的代码：

- **`TYPES`**：11 种人格。每种有 `style`（风格英文名）、`name`（人格标签）、`color`（霓虹主题色）、`desc`（一句话短描述）、`long`（结果页展开文案）。
- **`QUESTIONS`**：10 道题。每题 `tag`（主题词）、`text`（题干）、`options`（4 个选项）。
  - 每个选项：`primary` 主风格 **+2 分**，`secondary` 副风格 **+1 分**（key 用 `TYPES` 里的人格代号，如 `jazz`、`citypop`）。
  - 想让某个选项触发「贝壳病诊断书」彩蛋，加 `easterEgg:true`（目前在 Q6 的 D 选项）。

计分：10 题累加，取总分最高的人格；同分时取「先达到该分数」的（按 `TYPES` 声明顺序）。

底部分享图的群标识改 `GROUP_TAG` 常量。

## 加 / 减人格

在 `TYPES` 里加一条新人格（给个唯一代号），再在题目选项里把它配成某些选项的 `primary`/`secondary` 即可。建议每种人格做 3~4 次主风格，保证都能被答出来。
> 后续想扩到 15~20 种、融入群梗，凑齐群素材（高频词、名场面、人设标签）后照上面格式加就行。

## 老群友专属版（群梗认证）

第二套隐藏玩法：20 道群内梗知识题（有标准答案），按答对题数给称号。两个入口：

1. **结果页暗门**：答完常规音乐人格版，结果页底部有一道低调的「老群友专属」入口。
2. **开始页暗号**：在开始页**连点「群周年庆」那个徽章 3 下**直接进，不用先答常规版。

数据也在 `<script>` 数据区：

- **`QUESTIONS2`**：20 题。每题 `tag`（主题词）、`text`（题干）、`options`（3 个选项，纯字符串）。
- **`ANSWER_KEY`**：⚠️ **正确答案，必须填**。按题号顺序，每题一个字母 `A/B/C` 连成一行，共 20 个字符。例：`const ANSWER_KEY = "CCCBA...";`。**现在是占位的全 A，不换成真答案分数不准。**
- **`TITLES2`**：称号档位（满分 100 / 优秀≥80 / 合格≥60 / 潜水<60），名字、颜色、文案都可改。

得分 = 答对题数 ÷ 20 × 100（四舍五入）。
